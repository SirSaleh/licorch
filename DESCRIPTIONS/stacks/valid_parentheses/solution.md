# Valid Parentheses

To check if Parentheses are valid.

For exmaple: `"()"`, `"[]"`, `"{}"`, `"({[]})"` are valid but `"(]"`, `"[}"`, `"((("`, and `"{{{}}}}}}"` are not valid.
