# Fibonacci Sequence

To solve fibonnaci problem as it can describes as overlapping subproblems, dynamic programming is a good solution.

**Base Cases**: f(0) 0, f(1) = 1
**Transition Function**: f(n) = f(n-1) + f(n-2)
